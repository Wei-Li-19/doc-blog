# doc-blog

markdown编辑器博客原型实现，已实现编辑与文档管理的基础功能，提供Mac、Windows、Linux版本。
###### 当前功能
* markdown编辑器编辑、预览。
* 文档文件管理。
* 支持外部文件导入与自定义保存地址。

###### 后续功能
* 账号体系
* 文档云同步
* 文档发布为公开博客
* 移动端支持
